# 🚀 DSA Predictor | AI Instructor

An AI-powered Data Structures & Algorithms learning assistant built using Google Gemini API.  
This project acts like a smart DSA mentor that answers only DSA-related questions with a futuristic terminal-style UI.

---

# ✨ Features

- 🤖 AI-powered DSA Instructor
- 📚 Answers only DSA-related queries
- ⚡ Fast responses using Gemini API
- 🎨 Modern futuristic UI
- 💬 Interactive chatbot interface
- 🔒 System instruction protection
- 🌐 Built using HTML, CSS, JavaScript
